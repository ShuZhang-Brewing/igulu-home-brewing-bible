# Craft Beer

This directory contains all Craft Beer recipe chapters for the iGulu Home Brewing Bible.

## Sub-categories

- **Ales** — IPAs, Pale Ales, Wheat Beers, Belgian Styles, Stouts, Porters
- **Lagers** — Pilsners, Helles, Märzen, Dunkel
- **Sours** — Gose, Berliner Weisse, Lambic styles
- **Specialty** — Smoked, Fruit, Spiced, Barrel-aged

## File Naming

Use lowercase, hyphen-separated names:
- `west-coast-ipa.md`
- `belgian-tripel.md`
- `munich-helles.md`

See [TEMPLATE.md](../../TEMPLATE.md) for the standard chapter format.

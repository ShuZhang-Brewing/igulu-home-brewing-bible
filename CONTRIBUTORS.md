## How This File Works

Contributors are listed by tier and joining order. Within each tier, names are sorted by contribution date (earliest first). Once you've contributed, your name and basic info will be added here in the next release cycle.

If your name is missing or incorrect, please open an Issue labeled `contributors-update`.

## A Note on Inclusion

Every contributor — regardless of tier — is part of this project's story. Tiers exist to recognize sustained commitment, not to rank people. If you've ever helped, you belong here.

If you'd prefer not to be listed publicly, that's completely fine. Open an Issue labeled `privacy-preference` and we'll honor that.
